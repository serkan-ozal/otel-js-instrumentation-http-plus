# 0.0.4 (2023-12-01)

### Features

* Support masking HTTP request and/or response bodies over custom function(s)
  
<a name="0.0.3"></a>
# 0.0.3 (2023-11-29)

### Features

* Support tracing network timings by creating spans for the following traced network operations:
  - DNS Lookup
  - TCP Connect
  - TLS Handshake
  - TTFB (Time to First Byte)
  - Content Transfer
